# GameStats
GameStats is an extension plugin for [GameManager](https://github.com/GameManagementAPI/GameManager)
that provides comprehensive statistic tracking for individual players

### Features:
- Custom holograms
- Statistic command
- Statistic tracking

### Requirements
GameStats depends on GameManager and will not function without it.

➡️ Install the latest release of GameManager before starting the server:
https://github.com/GameManagementAPI/GameManager/releases/latest

### Installation

1. Download the [latest GameStats release](https://github.com/GameManagementAPI/GameStats/releases/latest).
2. Place the jar file into your server plugins folder.
3. Ensure GameManager is installed and configured.
4. Restart your server.
5. Edit `config.yml` to customize settings if needed.
